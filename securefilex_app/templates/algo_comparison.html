🔍 SecureFileX Functionality Overview

Explore the powerful features of our secure file management platform.
SecureFileX combines modern cryptography, steganography, and file sharing techniques with a visually rich interface to ensure your data remains private, secure, and easy to manage.
# Feature Description Access
1 File Encryption Encrypt files using AES-GCM, ChaCha20, Fernet, or RSA with user-specified keys or passwords. Logged-in Users
2 File Decryption Decrypt previously encrypted files using correct keys, metadata, or tags/nonces as needed. Logged-in Users
3 Secure Sharing Upload files and generate a short PIN-protected QR code link for others to download securely. Logged-in Users
4 Steganography Hide or extract secret messages inside image files using LSB encoding. Logged-in Users
5 Key Generation Dynamically generate secure cryptographic keys for different algorithms including RSA and ChaCha20. All Users
6 Algorithm Comparison View detailed comparisons (speed, strength, use cases) for supported algorithms via charts and tables. All Users
7 CyberChef Integration Launch an embedded CyberChef instance for additional encoding, decoding, and analysis operations. Logged-in Users
8 Audit Logging Track encryption, decryption, and sharing activities per user in secure logs (admin access). Admins Only
