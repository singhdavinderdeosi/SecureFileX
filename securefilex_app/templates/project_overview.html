🔐 SecureFileX

Secure Encryption, Decryption, and Sharing Platform

📌 Overview

SecureFileX is a powerful web-based tool built with Django that enables secure file encryption, decryption, and sharing using modern cryptographic algorithms. It offers a visually rich and accessible user interface with strong cryptographic backend support.

🚀 Key Features

  • Multi-Algorithm Support: Fernet, ChaCha20, AES-GCM, RSA
  • AI-Powered Algorithm Suggestions: Based on file type, size & sensitivity
  • QR Code & PIN Sharing: Secure link generation and QR access
  • Metadata + Key File Output: For reproducible decryption (.json + .keys)
  • Steganography: Hide and extract secret messages in images
  • Progressive Glassmorphism UI: Step-by-step wizard with binary rain background

🔧 Technology Stack

  • Backend: Django, Python,cryptography, PyCryptodome
  • Frontend: Bootstrap 5, AJAX, JavaScript, Canvas
  • Steganography: PIL (Python Imaging Library)

📂 Output File Structure

/media/encrypted/<username>/
  ├── <filename>.enc       # Encrypted file
  ├── <filename>.json      # Metadata
  ├── <filename>.keys      # Key file (optional)
  └── <filename>_qr.png    # QR code for sharing

🛡️ Security & Privacy

  • No permanent key storage on server
  • RSA keys provided as downloadable PEM files
  • User-specific encrypted/decrypted directories
  • PIN-based access for shared files

📈 Future Improvements

  • Auto-expiry of links and PINs
  • Cloud integration (e.g., S3)
  • File audit logging
  • Built-in virus scanning
