📄 SecureFileX Documentation

{% comment %} Section 1: Introduction {% endcomment %}

🔐 Introduction

SecureFileX is a modern, file security platform using Django. It provides tools for file encryption, QR/PIN-based sharing, and steganography, packaged in a simple and elegant UI for everyone.

{% comment %} Section 2: Features {% endcomment %}

🚀 Features

  • 🔒 Secure file encryption with Fernet, ChaCha20, AES-GCM, RSA
  • 🖼️ Steganography support to hide/reveal text in images
  • 🔑 Auto-generated secure key or passphrase support
  • 📄 Metadata stored in JSON format for integrity
  • 📦 PIN & QR-based file sharing system
  • 💬 AI-annotated logs and secure download links
  • 🎨 Responsive animated interface with drag-and-drop
{% comment %} Section 3: Installation {% endcomment %}

⚙️ Installation

git clone https://github.com/singhdavinderdeosi/SecureFileX.git
cd SecureFileX
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
{% comment %} Section 4: Usage {% endcomment %}

🧪 Usage

  • 🔐 Encrypt files by selecting an algorithm and uploading your file.
  • 🔓 Decrypt by providing the key, scanning QR, or uploading a `.keys` file.
  • 🖼️ Hide secret messages in images or extract them with steganography.
{% comment %} Section 5: Algorithms {% endcomment %}

🛡️ Encryption Algorithms

Algorithm Type Key/Nonce Details Use Case
Fernet Symmetric Key only (base64) Simple, secure file encryption
ChaCha20 Symmetric Key + Nonce High-speed stream cipher
AES-GCM Symmetric Key + Nonce + Tag Authenticated block encryption
{% comment %} Section 6: Metadata Format {% endcomment %}

📄 Metadata Format

Every encrypted file includes metadata in `.json` format:

{
  "algorithm": "AES-GCM",
  "key": "base64-encoded-key",
  "nonce": "base64-encoded-nonce",
  "tag": "base64-encoded-tag"
}
{% comment %} Section 7: Tech Stack {% endcomment %}

🛠️ Tech Stack

  • 💻 Backend: Django (Python)
  • 🔐 Crypto: cryptography, PyCryptodome
  • 🎨 Frontend: Bootstrap 5, AOS.js, Animate.css
  • 🖼️ Image Handling: Pillow
  • 🧪 Steganography: LSB pixel manipulation
