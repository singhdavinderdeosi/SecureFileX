# 🔐 SecureFileX

SecureFileX is a **Django-based secure file management system** that provides file encryption, decryption, image steganography, AI assistance, and secure user authentication. It is designed to protect sensitive files using modern cryptographic techniques.

---

## ✨ Features

- 🔒 AES-GCM File Encryption
- 🔓 Secure File Decryption
- 🖼️ Image Steganography
- 🤖 AI Assistant & File Summary
- 🔑 Automatic Key Management
- 👤 User Authentication
- 📂 Multi-file Upload & Download
- 📊 Activity Logs
- 🌐 Google & GitHub Login (Optional)

---

## 🛠️ Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite (Default)
- **Libraries:** Cryptography, Pillow, python-dotenv

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/singhdavinderdeosi/SecureFileX.git
cd SecureFileX
```

### 2. Create a Virtual Environment

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Apply Migrations

```bash
python manage.py migrate
```

### 5. Run the Server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```

### 6. Deactivate Virtual Environment

```bash
deactivate
```
